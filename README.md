# WXT + React

This template should help get you started developing with React in WXT.

## Main Technologies

- **WXT** (v0.19) - Browser extension framework for modern web development
- **React** (v19) - UI library
- **TypeScript** (v5.6) - Type-safe JavaScript
- **Redux Toolkit** (v2.5) - State management
- **Material UI** (v6.4) - Component library
- **Tailwind CSS** (v4.0) - Utility-first CSS framework
- **Emotion** (v11) - CSS-in-JS library (used by MUI)
- **Redux Chrome Storage** (v3.0) - Persistent state management for browser extensions
