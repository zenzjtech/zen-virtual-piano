## Sheet Mode Shortcuts

When a music sheet is loaded, these keyboard shortcuts are available:

| Key | Action |
|-----|--------|
| **Spacebar** | Play/Pause the sheet |
| **↑** | Increase tempo by 5 BPM (max: 240) |
| **↓** | Decrease tempo by 5 BPM (min: 40) |
| **Enter** or **→** | Next page (2-page spread) |
| **Backspace** or **←** | Previous page (2-page spread) |

## Tips for Playing

1. **Left-to-Right**: Keys progress naturally left to right across keyboard
2. **Shift for Sharps**: Hold Shift to play any black key (sharp/flat)
3. **Number Row**: Lowest octave starts with `1` = C2
4. **Home Row**: `asdfghjkl` provides easy access to mid-high range (B4-C6)
5. **Full Range**: From `1` (C2, 65Hz) to `m` (C7, 2093Hz)

## Memory Aids

- **Bass Notes**: Number row (1-0)
- **Middle Range**: Top letters (q-p) and home row (a-l)
- **High Notes**: Bottom row (z-m)
- **Sharps**: Just add Shift!
