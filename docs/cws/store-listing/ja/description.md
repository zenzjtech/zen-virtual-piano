# 禅ヴァーチャルピアノ - Chrome Web Store リスティング

## 説明

禅ヴァーチャルピアノでピアノ演奏の美しさを体験してください。プロフェッショナルグレードの仮想ピアノ機能を指先で実現する洗練されたブラウザ拡張機能です。

### ✨ 主な機能
- **複数の楽器**: グランドピアノ、クラシックピアノ、オルガンから選択可能。本格的なサウンドサンプル
- **美しいテーマ**: 禅、桜、神秘的なデザインを含む20以上の素晴らしいビジュアルテーマ
- **楽譜対応**: 自動ページめくりとテンポコントロールで楽譜を読み込んで演奏
- **録音・再生**: パフォーマンスを録音し、ダウンロード可能なファイルとして共有
- **キーボードショートカット**: カスタマイズ可能なショートカットでフルキーボード対応
- **レスポンシブデザイン**: すべての画面サイズとデバイスに最適化

### 🎹 こんな方に最適
- スケールや練習曲を練習する音楽学生
- 様々な楽器を探求するピアノ愛好家
- ピアノパフォーマンスを録音するコンテンツクリエイター
- いつでもどこでもピアノを演奏したい方

### 🔧 技術的なハイライト
- 複数のベロシティレイヤーを備えたリアルなピアノサウンドエンジン
- スムーズな60fpsアニメーションとトランジション
- 永続的な設定とユーザー設定
- クロスプラットフォーム対応（Windows、Mac、Linux）
- 外部ソフトウェア不要
