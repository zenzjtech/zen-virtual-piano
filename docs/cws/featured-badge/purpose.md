Purpose & Value Proposition

Question
What is the purpose of your extension? Describe the value it provides to Chrome users.

Answer
The Zen Virtual Piano extension provides a high-quality, realistic virtual piano instrument in the browser with 68 keys.  Its primary purpose is to offer an accessible, low-latency, and offline-capable rich musical tool for entertainment, practice, and education, featuring professional instruments, a vast sheet library, and advanced learning tools.
Value:
1. Instant Accessibility & Offline Use: Always available in the browser toolbar and works completely offline, enabling practice anywhere without internet.
2. Professional Musical Experience: 68-key keyboard with realistic sound and zero latency. Play via QWERTY mapping or mouse, supporting chords, polyphony, and visual feedback.
3. Multiple Instruments & Sound Options: 10+ professional instruments including piano variants, organ, strings, and more, with advanced controls like metron and sustain.
4. Educational Tools & Customization: Key Assist mode for visual guidance, record/playback for practice, 100+ themes, persistent settings.
