The extension is designed for easy, intuitive music creation and learning directly in the browser. Users can access it instantly by clicking the extension icon, which opens a dedicated piano interface.

1. Casual Entertainment: Relax and play favorite tunes or experiment with chords and melodies for fun, anytime without needing external software.
2. Music Practice and Learning: Use Key Assist mode to visually learn notes, practice scales, or follow built-in sheets. Ideal for beginners learning piano basics.
3. Composition and Creativity: Compose original melodies using multiple instruments, record performances, and export them. Switch between 10+ instruments for creative exploration.
4. Educational Purposes: Teachers can demonstrate musical concepts, while students practice interactively. Access 250+ built-in sheets or import from VirtualPiano.net.
5. On-the-Go Practice: Works offline, so musicians can practice anywhere with just their browser, using keyboard or mouse (portability)
